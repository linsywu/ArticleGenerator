# 风格蒸馏功能重构

## 目标

将蒸馏功能从账号详情弹窗中拆出，作为独立一级页面 `/distill`，提升管理效率和用户体验。

## 页面结构

左侧账号列表（可选中切换）| 右侧内容区（Tab 切换：参考文章 / 风格画像）

- 账号列表显示平台图标 + 账号名 + 风格画像状态标签
- 右侧参考文章 Tab：表格 + 添加/删除
- 右侧风格画像 Tab：展示画像内容 + 蒸馏按钮（含轮询进度）

## 交互流程

1. 进入页面 → 自动加载账号列表 → 默认选中第一个（如有）
2. 点击左侧账号 → 加载该账号的参考文章列表 + 风格画像
3. 添加参考文章 → 即时写入 → 刷新列表
4. 删除参考文章 → 确认弹窗 → 删除 → 刷新列表
5. 点击「蒸馏风格」→ 提交 Celery 任务 → 显示进度 → 完成后自动刷新风格画像
6. 蒸馏进行中时，按钮变为 loading 状态并提示等待

## 验收标准

- [ ] 侧边栏「风格蒸馏」入口可见
- [ ] 点击进入，左侧显示账号列表
- [ ] 选中账号后右侧显示参考文章
- [ ] 可添加/删除参考文章
- [ ] 点击蒸馏按钮触发生成任务
- [ ] 蒸馏完成/失败有反馈
- [ ] 风格画像自动刷新展示
- [ ] Console 无报错
- [ ] Network 无失败请求
