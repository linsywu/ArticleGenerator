# 千机笔内容中台系统 PRD V1.0

## 一、项目概述

### 1.1 项目背景

千机笔是一套面向自媒体矩阵运营的 AI 内容生产平台。

系统通过持续采集指定赛道公众号内容，构建内容知识库，并利用 AI 完成：

* 热点发现
* 爆文拆解
* 论点生成
* 风格蒸馏
* AI写作
* 多账号矩阵运营

最终帮助运营人员快速完成高质量内容创作与内容分发。

---

## 1.2 产品定位

定位：

```text
AI内容采集平台
+
AI内容分析平台
+
AI内容创作平台
+
AI账号风格训练平台
```

---

## 1.3 系统架构

```text

账号风格中心

文章创作中心
```

# 五、采集执行流程

## 5.1 技术方案

公众号后台登录态采集方案

```text
searchbiz
    ↓
获取fakeid

appmsg
    ↓
获取文章列表

获取文章链接

解析文章HTML

提取正文

图片转存

内容清洗

Markdown转换

内容Hash生成

文章入库

AI标签生成
```

---

## 5.2 核心接口

### 搜索公众号

```http
GET /cgi-bin/searchbiz
```

作用：

```text
公众号名称
↓
获取fakeid
```

---

### 获取文章列表

```http
GET /cgi-bin/appmsg
```

作用：

```text
fakeid
↓
获取公众号历史文章列表
```

返回：

```json
{
  "app_msg_list":[]
}
```

---

## 5.3 采集器架构

```text
Collector Scheduler
    ↓
Collector Worker
    ↓
Mp Client
    ↓
Parser
    ↓
Storage
```

---

### Scheduler

任务调度器

负责：

* cron执行
* 队列投递

---

### Worker

采集执行器

负责：

* 调用公众号接口
* 拉取文章

---

### Parser

内容解析器

负责：

* HTML解析
* 图片处理
* Markdown转换

---

### Storage

存储层

负责：

* 入库
* OSS转存
