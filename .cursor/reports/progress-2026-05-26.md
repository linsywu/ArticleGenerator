## 二、文章生成 Pipeline 优化 ✅ 完成

### 2.1 去 AI 味链式处理
- 新增 `humanize` 场景：生成后自动检测 AI 痕迹并真人化重写
- 全链路：生成 → 去AI味重写 → 质量评审 → 合规评审
- humanize 策略：打破工整句式、删除过渡词、插入思维跳跃、留白处理

### 2.2 生成参数优化
- temperature: 0.8 → 0.95
- 新增 frequency_penalty: 0.4
- 新增 presence_penalty: 0.4
- 新增 top_p: 0.9

### 2.3 System Prompt 重构
- 禁止 chatbot 响应（"好的""收到""以下是"）
- 禁止 Markdown 格式
- 结构多样性要求：禁止每次用同样开头
- 留白要求：道理只讲七分，禁止总结升华
- 段落节奏要求：长短参差，可一句成段

### 2.4 User Prompt 增强
- 从裸标题改为结构化写作指令
- 随机开头方式（观点直出 / 对话 / 个人经历 / 数据）
- 明确的留白和反模板要求
