素材文章 (mp_materials)
    │
    ├─ 用户点击「💡 创作方向」
    │       │
    │       ↓
    │  POST /api/articles/{material_id}/argument-storm
    │       │  → LLM Gateway (scenario=argument_storm)
    │       │  → 输入：文章 title + content_markdown（若未解析，先触发解析）
    │       │  → 输出：5 个创作方向 JSON
    │       │       │
    │       │       ↓
    │       │  前端展示创作方向 Modal
    │       │
    │       ├─ 「一键带入创作」→ 跳转 CreateView，预填：
    │       │    - 标题、目标人群、核心观点、内容大纲
    │       │    - 风格模板（默认账号风格）
    │       │    - 参考素材（原文）
    │       │
    │       └─ 保存结果到 analysis_reports 表（可回溯）
