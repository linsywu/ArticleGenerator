子模块 3.2：采集任务管理

字段类型说明
idbigint主键
namevarchar(200)任务名称
credential_idbigint FK使用的采集凭证
track_idsjson目标赛道
account_idsjson目标公众号（空=按赛道全采）
collect_modevarchar(30)history_50/100/200 / date_range / incremental
schedule_typevarchar(20)manual / interval / cron
cronvarchar(50)cron 表达式（schedule_type=cron 时）
statusvarchar(20)idle / running / paused / error

采集模式

📥 历史同步
  • 最近 50 篇
  • 最近 100 篇
  • 最近 200 篇
  • 指定时间范围
📥 增量同步
  • 仅同步新文章
  • 基于 last_collect_time
  • 基于 URL + content_hash 去重
⏰ 执行方式： 定时任务 (cron) 间隔任务 (每N小时) 手动执行
