采集执行流程

┌──────────────────────────────────────────────────────────────────┐
│                    🎯 Collector Scheduler                         │
│  Celery Beat / APScheduler → 扫描 collect_task 表                │
│  匹配 cron/interval → 投递到 Worker Queue                        │
└───────────────────────────┬──────────────────────────────────────┘
                            │
┌───────────────────────────┴──────────────────────────────────────┐
│                    🔧 Collector Worker                            │
│  Celery Worker → 逐个公众号 → 调用 MpClient                       │
│  ┌──────────────────────────────────────────────────────────┐   │
│  │                  🌐 MpClient (HTTP)                       │   │
│  │  GET /cgi-bin/searchbiz?query=名称&token=xxx              │   │
│  │    → 获取 fakeid（如未获取过）                              │   │
│  │  GET /cgi-bin/appmsg?fakeid=xxx&begin=0&count=10         │   │
│  │    → 获取文章列表 (app_msg_list)                           │   │
│  └───────────────────┬──────────────────────────────────────┘   │
└──────────────────────┼───────────────────────────────────────────┘
                       │
┌──────────────────────┴───────────────────────────────────────────┐
│                    📄 Parser                                      │
│  对每篇文章：                                                      │
│  1. HTTP GET 文章链接 → HTML                                      │
│  2. 选择器 #js_content → 提取正文                                 │
│  3. 提取：标题、摘要、封面、作者、发布时间                           │
│  4. 图片：data-src → 下载 → OSS转存 → 替换URL                     │
│  5. Turndown (HTML→Markdown)                                     │
│  6. SHA256(content) → content_hash                               │
└──────────────────────┬───────────────────────────────────────────┘
                       │
┌──────────────────────┴───────────────────────────────────────────┐
│                    🗄️ Storage                                     │
│  去重检查：                                                        │
│  ├─ original_url 唯一索引（URL去重）                               │
│  ├─ content_hash 对比（内容去重）                                  │
│  └─ 标题相似度 → 跳过已存在                                       │
│  入库 mp_materials 表                                             │
└──────────────────────────────────────────────────────────────────┘

🔴 技术难点预警
  1. 公众号接口反爬：appmsg 接口有频率限制，需要请求间隔控制
  2. HTML 解析鲁棒性：文章页结构可能变化，#js_content 不是100%可靠
  3. 图片转存：微信图片可能有防盗链，OSS 需要正确处理 referer
  4. 凭证过期检测：需要主动探测 token/cookie 有效性，及时预警
  5. 大量文章解析：单篇文章解析耗时可观，需要并发控制
