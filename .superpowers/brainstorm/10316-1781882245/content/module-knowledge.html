数据模型

库表名核心字段提取内容
💡 观点库 knowledge_opinions id, material_id, content(text), category, keywords(json), created_at 核心观点、支撑论据、结论
📊 案例库 knowledge_cases id, material_id, name(varchar), type(人物/企业/产品/数据), description(text), created_at 人物、企业、产品、数据
✨ 金句库 knowledge_quotes id, material_id, content(text), style(varchar), created_at 精彩表达、金句
📝 标题库 knowledge_titles id, material_id, title(varchar), structure(varchar), keywords(json), created_at 标题模板、标题结构

批量提取流程

🤖 AI 批量提取
  1. 用户在素材中心选中多篇文章
  2. 点击「提取到知识库」
  3. 选择目标库（观点/案例/金句/标题 可多选）
  4. 调用 LLM Gateway 逐篇提取
  5. 结构化结果自动入库
✍️ 手动添加
  1. 用户在对应库中点击「手动添加」
  2. 填写内容和分类信息
  3. 可选关联来源文章
  4. 入库后可在创作时检索
📌 与创作中心的关系：

创作文章时，侧边栏可检索知识库：找相关观点支撑文章论点、引用案例增加说服力、插入金句提升可读性、借鉴标题模板优化标题。四库是创作的"弹药库"。
